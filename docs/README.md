## Apa itu Markdown (.md)?

**Markdown** adalah bahasa markup ringan yang digunakan untuk memformat teks dengan sintaks yang mudah dibaca dan ditulis. File dengan ekstensi `.md` adalah file Markdown.

### Keuntungan Markdown:
- âœ… Mudah dibaca dan ditulis (plain text)
- âœ… Dapat dikonversi ke HTML, PDF, dan format lainnya
- âœ… Digunakan secara luas di GitHub, GitLab, dan platform dokumentasi lainnya
- âœ… Mendukung formatting seperti **bold**, *italic*, headers, lists, code blocks, dll.

### Contoh Sintaks Markdown:
```markdown
# Heading 1
## Heading 2
**Bold text**
*Italic text*
- List item
[Link](url)
```

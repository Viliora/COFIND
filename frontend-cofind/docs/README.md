# Dokumentasi Frontend

Folder ini berisi dokumentasi spesifik untuk frontend COFIND.

## File-file Dokumentasi

### Development Guides
- **QUICK_START_DEV.md** - Panduan cepat untuk memulai development
- **DEVELOPMENT_OPTIMIZATION.md** - Optimasi untuk development
- **HMR_SETUP.md** - Setup Hot Module Replacement (HMR)

### Feature Guides
- **HERO_SWIPER_GUIDE.md** - Panduan penggunaan Hero Swiper component
- **ENABLE_API_PHOTOS.md** - Cara enable foto dari API
- **IMAGE_OPTIMIZATION.md** - Optimasi gambar dan komponen OptimizedImage
- **README_CACHE.md** - Dokumentasi sistem cache

## Catatan
- README.md di root `frontend-cofind/` tetap di root sebagai dokumentasi utama project
- Dokumentasi spesifik feature/component ada di folder ini
